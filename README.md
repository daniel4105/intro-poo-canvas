# Intro Canvas 2D con JavaScript y OOP

Aplicación web desarrollada con **JavaScript y HTML5 Canvas** que demuestra la creación y renderizado de objetos gráficos utilizando **Programación Orientada a Objetos (OOP)**.

El proyecto genera círculos dentro de distintos elementos **canvas**, mostrando ejemplos de:

* Creación de objetos gráficos mediante clases
* Posiciones aleatorias dentro de un canvas
* Generación de múltiples objetos
* Colores generados aleatoriamente
* Uso de arreglos para almacenar objetos

---

## Vista del proyecto

La aplicación muestra **tres canvas principales**:

### 1️⃣ Objeto 2D mediante OOP

Se crea una instancia de la clase `Circle` y se renderiza en el centro del canvas utilizando el método `draw()`.

### 2️⃣ Objeto con posición aleatoria

Se genera un círculo con:

* Posición aleatoria
* Radio aleatorio
* Colores aleatorios

Asegurando que el círculo **no se salga del canvas**.

### 3️⃣ Múltiples objetos

Se generan varios círculos mediante un ciclo `for`.
Cada círculo:

* Tiene tamaño aleatorio
* Tiene colores aleatorios
* Se almacena dentro de un arreglo
* Se renderiza dentro del canvas
